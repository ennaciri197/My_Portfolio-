Projets Hybrides — IoT, DevOps & Systèmes Connectés

Projet Majeur

Temperature Dashboard

Supervision IoT en Temps Réel

Projet full-stack qui fait le pont entre le matériel, le DevOps, la base de données et le web. Un capteur DHT11 connecté à un Raspberry Pi 3B collecte température et humidité via GPIO, les données transitent dans une stack Docker Compose (MariaDB ARMv7 + PHP), et un dashboard web affiche les courbes en temps réel grâce à Chart.js et des requêtes AJAX toutes les 5 secondes.
